Foundations

Signifiers

Signifiers define the visual cues that communicate what an element means, what state it is in, and what users can do with it.

기표는 사용자가 UI를 보고 해당 요소의 역할, 상태, 가능한 행동을 이해할 수 있도록 하는 시각적 단서입니다. 이 페이지는 디자인 시스템 안에서 특정 시각 표현이 특정 의미에만 사용되도록 제한하는 공통 계약을 정의합니다.

Overview

Signifier 페이지의 목적은 UI의 시각 표현이 어떤 의미, 상태, 행동 가능성을 나타내는지 일관되게 정의하는 것입니다. 기표는 장식이 아니라 사용자가 인터페이스를 해석하는 방식에 직접 관여하는 규칙입니다.

예를 들어 primary color text, underline, pointer cursor, external link icon, check icon은 각각 특정한 의미를 갖습니다. 이러한 단서가 다른 의미로 재사용되면 사용자는 요소의 역할을 오해할 수 있습니다.

Core Principles

One signifier, one primary meaning

하나의 시각적 단서는 시스템이 명시적으로 허용하지 않는 한 다른 의미로 재사용하지 않습니다. 예를 들어 외부 링크 아이콘은 외부 목적지로 이동한다는 의미에 사용하며, 일반적인 내부 이동이나 상세 보기에는 사용하지 않습니다.

Do not use interaction cues as decoration

클릭 가능성을 암시하는 색상, 밑줄, 커서, 아이콘은 장식이나 일반 강조로 사용하지 않습니다. 사용자는 이러한 단서를 인터랙션 가능성으로 학습하기 때문입니다.

Do not rely on color alone

상태나 피드백은 색상만으로 전달하지 않습니다. 중요한 상태는 아이콘, 텍스트, 형태, 위치, ARIA 속성 등과 함께 제공해야 합니다.

Keep signifiers consistent across components

컴포넌트의 접근성 속성은 다를 수 있지만 사용자가 인지하는 시각적 기표는 일관되게 유지합니다. 예를 들어 Tabs는 aria-selected, Toggle Button은 aria-pressed, Checkbox는 aria-checked를 사용할 수 있지만 선택 상태의 시각적 단서는 공통된 원칙을 따릅니다.

Clickable & Link Signifiers

Clickable signifier는 사용자가 특정 요소를 누르거나 이동할 수 있음을 나타냅니다. primary color text, underline, pointer cursor, button-like shape은 인터랙티브 요소에만 사용합니다.

Primary color text

Primary color text는 링크, 텍스트 버튼, 인라인 액션처럼 클릭 가능한 텍스트에 예약합니다. 본문 강조, 메타 정보 강조, 장식적 하이라이트에는 사용하지 않습니다.

Do

<a href="/settings">설정으로 이동</a>

Don’t

<p>이 기능은 <span class="text-primary">중요합니다</span>.</p>

External links

외부 링크 아이콘은 현재 제품 또는 서비스 맥락을 벗어나는 목적지를 나타냅니다. 일반적인 내부 이동, 패널 확장, 상세 보기에는 사용하지 않습니다.

Icon Indicators

Icon indicator는 장식이 아니라 특정 의미를 전달하는 시각적 단서입니다. 아이콘을 indicator로 사용하는 경우, 해당 아이콘의 의미와 사용 범위를 명확히 고정해야 합니다.

External link
외부 목적지로 이동하거나 새 외부 맥락을 여는 경우에 사용합니다.
Check
선택됨, 완료됨, 확인됨을 나타냅니다. 단순한 긍정 장식으로 사용하지 않습니다.
Warning
사용자의 주의가 필요한 상태를 나타냅니다. 마케팅 강조 용도로 사용하지 않습니다.
Error
오류, 실패, 수정이 필요한 상태를 나타냅니다.
Info
보조 정보나 설명 가능한 상태를 나타냅니다.
Download
파일 또는 데이터를 내려받는 액션에 사용합니다.
More
추가 액션 메뉴 또는 숨겨진 옵션을 나타냅니다.
Required
필수 입력 또는 필수 조건을 나타냅니다.

Selected Indicators

Selected는 여러 선택지 중 사용자가 선택한 항목을 나타냅니다. 컴포넌트별 의미론적 상태는 다를 수 있지만, 선택 상태를 전달하는 시각적 기표는 일관되게 관리합니다.

Related states

Selected
여러 선택지 중 선택된 항목입니다.
Current
현재 위치나 현재 보고 있는 항목입니다.
Active
현재 활성화되어 있는 항목입니다.
Checked
체크박스, 라디오, 메뉴 옵션의 체크 상태입니다.
Pressed
토글 버튼이 눌린 상태입니다.

Visual cues

  • selected background
  • selected border
  • check icon
  • stronger label weight
  • selected indicator line

Component mapping

Tabs
aria-selected를 사용합니다.
Toggle Button
aria-pressed를 사용합니다.
Radio
aria-checked 또는 native checked state를 사용합니다.
Checkbox
aria-checked 또는 native checked state를 사용합니다.
Menu option
역할에 따라 aria-checked 또는 aria-selected를 사용합니다.

Interaction States

Interaction states는 사용자가 요소와 상호작용하는 과정에서 현재 상태를 이해하도록 돕는 시각적 단서입니다.

Hover
포인터가 요소 위에 위치해 있으며 상호작용 가능함을 나타냅니다.
Focus
키보드 또는 프로그램적 탐색으로 요소가 포커스를 받은 상태입니다.
Focus-visible
키보드 탐색 등 명시적 포커스 표시가 필요한 상태입니다.
Pressed
요소가 눌리는 중이거나 토글된 상태입니다.
Disabled
현재 사용할 수 없는 상태입니다. 단순히 시각적으로 약하게 보이는 것 이상의 의미를 갖습니다.
Loading
작업이 진행 중이며 결과가 아직 확정되지 않은 상태입니다.

Feedback Signifiers

Feedback signifier는 사용자의 행동이나 시스템 상태에 대한 결과를 전달합니다. success, info, warning, error는 색상뿐 아니라 명시적 메시지와 접근성 관계를 함께 제공해야 합니다.

Error feedback

Error feedback은 다음 요소를 함께 제공해야 합니다.

  • error tone
  • error icon when helpful
  • explicit message text
  • affected field or region
  • accessible relationship such as aria-invalid and aria-describedby

Feedback types

Success
작업이 성공적으로 완료되었음을 나타냅니다.
Info
추가 정보나 참고 가능한 상태를 나타냅니다.
Warning
진행 전에 주의가 필요한 상태를 나타냅니다.
Error
실패, 오류, 수정 필요 상태를 나타냅니다.
Empty
데이터나 결과가 없는 상태를 설명합니다.
Progress
작업이 진행 중임을 나타냅니다.

Structure & Grouping

Layout 자체를 정의하기보다, 공간과 구조가 의미를 전달하는 경우를 다룹니다. 같은 영역에 배치된 요소는 같은 개체나 맥락에 속한다는 신호가 될 수 있고, 구분선은 서로 다른 항목의 경계를 나타낼 수 있습니다.

Grouping
같은 컨테이너 안의 요소는 같은 맥락 또는 개체에 속함을 나타냅니다.
Separation
구분선, 간격, 배경 차이는 항목이나 섹션의 경계를 나타냅니다.
Hierarchy
크기, 위치, 간격, 밀도는 정보의 우선순위를 전달합니다.
Relationship
인접성, 정렬, 포함 관계는 요소 간 관계를 나타냅니다.
Density as meaning
높은 밀도는 관리형/목록형 인터페이스를, 낮은 밀도는 탐색형/강조형 인터페이스를 암시할 수 있습니다.
