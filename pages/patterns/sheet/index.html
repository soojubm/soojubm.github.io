Modal vs Non-modal Presentation 레이어는 사용자의 주의를 얼마나 차단하느냐에 따라 두 가지로 나뉜다. 이 기준은 시각적 형태(Sheet, Dialog 등)가 아니라 배경과의 상호작용 허용 여부 로 결정된다. Modal 배경과의 상호작용을 차단하고 사용자의 즉각적인 응답을 요구한다. Backdrop(dim)이 뒤를 덮고, 포커스는 레이어 내부에 갇힌다(focus trap). 닫기는 명시적인 버튼 액션으로만 허용하는 것이 원칙이며, 배경 클릭으로 닫는 기능은 중요도가 낮은 작업에서만 예외적으로 허용한다. Non-modal 배경과의 상호작용을 허용한다. 현재 작업 맥락을 유지하면서 부가 정보나 서브태스크를 제공할 때 사용한다. 포커스 트랩이 없고, 외부 클릭·ESC로 언제든 닫을 수 있다. Sheet variant별 분류 Mozilla Popover API HIG action-sheets alertdialog role sheet-body는 콘텐츠가 넘치면 내부에서 스크롤됩니다. mm-sheet에 height를 지정하면 고정 높이 내에서 body가 스크롤됩니다.
