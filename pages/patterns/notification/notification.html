소음. 목표. 우선순위. 중요한 문제에 집중. 비판적인 관점. 정보 필터링.

디지털 프로덕에 엄격한 그리드 시스템은 소음이다. 디자인 이론에 근거한 그리드 시스템, 예를 들면 12칼럼 그리드를 사용하면 코드 환경에서 사용자의 다양하고 예측 불가능한 화면 크기에 대응하기 어렵다. 그리드 시스템은 디자이너에게만 유효하게 동작한다. 장점이 없음. 2~4개의 칼럼을 가지는 레이아웃 케이스와 간격 시스템을 정의하고 가로 스크롤을 활용. 대안 솔루션도 작성하가.

디자인시스템이라는 용어 자체가 소음이다. 스타일가이드 또는 유아이라이브러리가 디자인시스템으로 불린다.

  • popover: 전체 알림 목록을 볼 수 있는 페이지 필요. 모두보기 링크 버튼 필요.
  • modal: 전체 알림 목록을 볼 수 있는 페이지 불필요. 모달에서 infinite scroll이나 more load
  • 개별 알림 액션: 읽음-링크/삭제
  • MVP를 위한 팁: 전체를 하나의 클릭 영역으로 정의하고, 콘텐츠와 작성자를 분리하지 않고 하나의 텍스트 더미.
