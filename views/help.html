✔신청했는데 뉴스레터가 안 와요.

뉴닉과 고슴이가 프로모션함이나 스팸함에 빠지지 않도록, 잠시만 시간을 내주세요!😉


지메일로 신청하셨다고요?

  1. 검색창 오른쪽의 검은 화살표(▼)를 클릭합니다.
  2. 보낸 사람에 'whatsup@newneek.co'를 넣고, '필터 만들기' 버튼을 클릭합니다.
  3. 그 메일이 도착했을 때의 '수행 작업'을 고를 수 있는데요. '절대 스팸으로 신고하지 않음'을 체크한 후 '다음 카테고리로 분류'에서 '기본'을 선택합니다.
  4. 마지막으로 '필터 만들기' 누르면 끝!

네이버메일로 신청하셨다고요?

  1. 검색창 오른쪽의 검은 화살표(▼)를 클릭합니다.
  2. 보낸 사람에 'whatsup@newneek.co'를 넣고, '필터 만들기' 버튼을 클릭합니다.
  3. 그 메일이 도착했을 때의 '수행 작업'을 고를 수 있는데요. '절대 스팸으로 신고하지 않음'을 체크한 후 '다음 카테고리로 분류'에서 '기본'을 선택합니다.
  4. 마지막으로 '필터 만들기' 누르면 끝!

회사 메일로 신청하셨다고요?

  1. 일단 스팸 메일함을 확인해 주세요.
  2. 차단 목록도 확인 부탁드려요.
  3. 마지막으로 whatsup@newneek.co를 주소록에 추가해 주세요.
  4. 그런데도 메일함에서 고슴이를 볼 수 없다면, 회사 이메일 서버가 뉴닉을 차단하는 것이랍니다. 안타깝게도 이 경우에는 저희가 도와드리기 어려운 점 양해 부탁드리며, 자주 사용하는 개인 이메일로 다시 한 번 구독 신청 권장드릴게요. 🦔

✔ 다른 이메일로 뉴스레터 받아보고 싶어요.

죄송하게도 현재는 뉴스레터 하단에 있는 '수신거부' 버튼을 통해 기존 이메일을 수신 거부한 후 새로운 이메일로 다시 구독하는 방법이 유일해요. 조금 더 편리한 방법을 만드는 중이니, 조금만 기다려 주세요.

✔해지하고 싶어요. (정말요?! 😢)

뉴닉 서비스 해지는 모든 뉴스레터 하단에 있는 '수신거부' 버튼을 통해 하실 수 있답니다. 더욱더 나은 뉴스레터로 찾아뵐 수 있도록 노력할게요. 그동안 건강하셔야 해요!
