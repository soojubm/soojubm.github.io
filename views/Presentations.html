Modal

작성 중.

모달이 될 수 있는 컴포넌트.

  • Dialog
  • Page - portfolio work / width 정의하기
  • Multiple filters
  • Dropdown menu
  • 사용자의 워크플로를 완료하는 데 필요한 사용자 입력을 요청 input. 일반적으로 편집 또는 관리 작업과 같은 짧고 자주 발생하지 않는 작업에 사
  • 사용자가 모달을 해제할 때까지 이전 워크플로로 돌아갈 수 없습니다
  • 사용자의 인터렉션이 필요한 경우
  • 이전 페이지와 맥락을 유지하는 콘텐츠 표시
  • 사용자가 직접 모달을 열도록 트리거.
  • 모바일에서 페이지가 될 수 있다. airbnb case: 모달일 때 x 버튼. 모바일에서 페이지처럼 보이면서 arrow 버튼으로 변경됨
  • 닫기 버튼을 왼쪽에 배치하는 케이스와 오른쪽에 배치하는 케이스를 공부해보기.
  • 모달 보기의 작업 이름을 지정하는 제목 또는 작업을 설명하거나 지침을 제공하는 추가 텍스트를 제공하면 사람들이 앱에서 자신의 위치를 ​​유지하도록 도울 수 있습니다.

  • click. dim / overay

  • click. 작업완료 / 취소

  • body element의 스크롤을 허용할 것인가.
